## 학습한 내용 
* line-height
  * 줄 간격을 설정함
  * 폰트 사이즈에 맞게 조정되어야 하기 때문에 상대적인 값을 사용
  * 상속을 고려해 단위를 생략하는게 좋음
  * 단위를 생략하고 숫자만 쓰면 폰트 사이즈에 숫자를 곱한 값으로 설정 됨
